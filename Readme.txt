Smart Sorting Web Application
----------------------------

This is a Flask-based web app for detecting Fresh vs Rotten fruits and vegetables
using a trained deep learning model (healthy_vs_rotten.h5).

Steps to run:
1. Install requirements:
   pip install flask tensorflow opencv-python numpy

2. Run app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

Upload an image and get prediction result.
